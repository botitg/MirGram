## 📊 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### Архитектура:

- **Backend**: Node.js + Express
- **Database**: SQLite3
- **Real-time**: Socket.io
- **Frontend**: Vanilla JavaScript
- **Auth**: JWT tokens + bcryptjs

### API Endpoints:

- `POST /api/auth/register` - регистрация
- `POST /api/auth/login` - вход
- `GET /api/auth/me` - текущий пользователь
- `GET /api/chats` - список чатов
- `GET /api/chats/:chatId/messages` - сообщения
- `POST /api/chats/:chatId/messages` - отправка

### Socket.io Events:

- `message:new` - новое сообщение
- `member:updated` - обновление участника
- `connect` / `disconnect` - статус соединения

## 🛠️ РЕШЕНИЕ ПРОБЛЕМ

### Проблема: "Я все еще выбрасываюсь на auth при перезагрузке"

**Причина**: localStorage очищен или токен истек

**Решение**:

1. Откройте DevTools (F12)
2. Перейдите на вкладку "console"
3. Введите: `localStorage.getItem('mirno.token')`
4. Если результат `null`, токена нет
5. Попробуйте войти заново

---

### Проблема: "Я не вижу сообщения других пользователей"

**Причина**: Socket.io не подключен или истекло соединение

**Решение**:

1. Проверьте консоль (F12 → console)
2. Должно быть сообщение "Подключено 🌐"
3. Если нет, проверьте интернет
4. Переезагрузите страницу (Ctrl+R)

---

### Проблема: "Я не вижу чаты"

**Причина**: Чаты не загрузились из БД

**Решение**:

1. Проверьте консоль на ошибки (F12 → console)
2. Попробуйте разлогиться и войти заново
3. Если не помогает, очистите БД: `rm data/mirnachat-online.db`
